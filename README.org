* Overview

/bisos.uploadAsCs/ provides various facilities for uploading (importing) a module into the
pyCS framework and execute that module.

* A Standalone Piece of BISOS --- ByStar Internet Services Operating System

Layered on top of Debian, [[https://github.com/bisos][BISOS]]: (By* Internet Services Operating System) is a
unified and universal framework for developing both internet services and
software-service continuums that use internet services. See [[https://github.com/bxGenesis/start][Bootstrapping
ByStar, BISOS and Blee]] for additional information.
*BISOS* is a foundation for [[https://github.com/mohsenBanan][The Libre-Halaal ByStar Digital Ecosystem]] which is
described as a cure for losses of autonomy and privacy in a book titled: [[https://github.com/bxplpc/120033][Nature
of Polyexistentials]].

bisos.loadAsCs is a standalone piece of BISOS. It can be used as a self-contained
Python package separate from BISOS. Follow the installation and usage
instructions below for your own use.

* Installation of bisos.uploadAsCs Package

The sources for the bisos.loadAsCs pip package are maintained at:
https://github.com/bisos-pip/uploadAsCs.

The bisos.uploadAsCs pip package is available at PYPI as
https://pypi.org/project/bisos.uploadAsCs

You can install bisos.loadAsCs with pip or pipx.

** Installation With pip

If you need access to bisos.uploadAsCs as a python module, you can install it with pip:

#+begin_src bash
pip install bisos.uploadAsCs
#+end_src

See below for this packages list of commands.

** Installation With pipx

If you only need access to bisos.uploadAsCs on command-line, you can install it with pipx:

#+begin_src bash
pipx install bisos.uploadAsCs
#+end_src

* Usage of bisos.uploadAsCs Package

Run:

#+begin_src bash
uploadAsCs.cs
#+end_src

Without any parameters and arguments, it gives you a menu of
common invokations.
